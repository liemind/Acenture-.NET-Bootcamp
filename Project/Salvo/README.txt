# Salvo

Salvo es un programa creado para el Curso de .NET ACCENTURE 2021.

## Instalación

Abra el Archivo utilizando Visual Studio 2019 o Visual Studio Code.

## Cómo ejecutarlo.
### En VS 2019
Abra el archivo y coloque play.

### En VS Code

En la consola, ejecutamos:
```
dotnet build
dotnet run -p Salvo.sln
```
